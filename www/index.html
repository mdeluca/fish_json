Rapala - X-Rap

Features

  • Slashbait® Action
  • Suspending on Pause
  • Textured Translucent Body

The X-Rap® is "all about the action." The versatility of the X-Rap allows an angler to impart a number of fish-catching actions in a single retrieve. Casts like a bullet for long accurate casts.

Prominent scales and lateral line on the fuselage capture and flash light like a beacon. It can be fished both in an aggressive "Slashbait®" technique or with a classic Rapala® wobble.

The angler defines the cadence for maximum effectiveness. Loaded with features from tip to tail, textured translucent body, internal holographic foil, internal long cast system (XR04 excluded), 3D holographic eyes, flash foil teaser tails, Slashbait® action, Rapala "wobble" action, suspends on pause. All topped off with premium VMC® black nickel hooks, each hand-tuned and tank-tested to ensure top performing action from the start.

Running Depth

2' - 3'

Body Length

1-1/2"
